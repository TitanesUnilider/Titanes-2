Engineering materials
====

This repository contains engineering materials of a self-driven vehicle's model participating in the WRO Future Engineers competition in the season 2022.

## Content

* `t-photos` contiene dos fotos del equipo (una seria y la otra graciosa)
* `v-photos` contiene 6 fotos del vehiculo (de diferentes lados)
* `video` contiene un video del vehiculo avanzando autonomamente
* `schemes` contiene fotos y esquemas de los componentes que componen al vehiculo
* `src` contiene el codigo utilizado para el vehiculo,(originalmente en EV3)

## Introduction

_This part must be filled by participants with the technical clarifications about the code: which modules the code consists of, how they are related to the electromechanical components of the vehicle, and what is the process to build/compile/upload the code to the vehicle’s controllers._
